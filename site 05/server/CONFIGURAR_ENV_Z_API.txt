IMPORTANTE:
- O Client-Token precisa estar configurado no painel Z-API
- Acesse: https://developer.z-api.io
- Configure o token na aba "Segurança" da instância
- Ative/Habilite o token

Depois de configurar no painel, teste com:
node testar-z-api.js
